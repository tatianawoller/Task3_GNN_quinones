# usage

## requirements

- miniforge or miniconda installation
  
## installation

```bash
git clone git https://github.com/tatianawoller/Task3_GNN_quinones.git
cd https://github.com/tatianawoller/Task3_GNN_quinones
conda env create -f environments.yaml
conda activate horta_aimd_tw
```

- does not require gpu
